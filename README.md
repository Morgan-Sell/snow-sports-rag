# snow-sports-rag
RAG application that retrieves data from synthetic ski and snowboarding industry dataset.



# Data Explained

The synthetic dataset was generated by Claude is intended represent the sample of data the corresponds to the professional ski and snowboard industries. There are five document types:
1. Profession Circuits & Governing Bodies
2. Athletes
3. Ski Resorts
4. Competitions
5. Results

The benefit of this dataset is that there are many dependencies across the datasets, presenting an ideal use case for RAG. The diagram below visualizes the dependencies.

![Documents Diagram](img/snow_sports_data_diagram.png)

## Development

- Install: [uv](https://docs.astral.sh/uv/) — `uv sync --group dev` from the repo root (includes pytest and ruff; matches CI); run tools with `uv run …`.
- **Tests:** `uv run pytest` (default: excludes tests marked `integration` — no heavy models or network).
- **Integration tests:** `uv run pytest -m integration` (Chroma on disk, sentence-transformers, etc.).
- **Eval CLIs:** `uv run snow-sports-rag-sweep`, `uv run snow-sports-rag-trace-analyze --help` (see [docker/README.md](docker/README.md) for container use).
